# Project: Chain Lens Documentation Processing and Translation

## Overall Goal
Download all documentation pages from the Chain Lens website (https://docs.chainlens.net/).
Save the original English content of each page as a Markdown file in the 'docs/User/en/' directory relative to project root.
Translate the content of each page into Vietnamese.
Save the translated Vietnamese content of each page as a Markdown file in the 'docs/User/vi/' directory relative to project root.
